# 🧪 Master Prompt — Chemical Manufacturer Website (Gujarat, India)

---

## COMPANY PROFILE

- **Industry**: Industrial & Specialty Chemical Manufacturing
- **Location**: Gujarat, India (a globally recognized chemical manufacturing hub)
- **Experience**: 5–10 years in the industry
- **Products**: 15+ industrial and specialty chemicals
- **Certifications**: ISO Certified
- **USP**: Reliable quality, competitive pricing, export-ready supply chain, ISO-backed manufacturing

---

## PRIMARY GOAL

The #1 objective of this website is to **generate global export/import inquiries** from:
- 🇺🇸 USA & Canada
- 🌍 Middle East & Africa
- 🌏 Southeast Asia & Far East

Every section must be designed with this conversion goal in mind — build trust fast and make it frictionless to submit an inquiry.

---

## DESIGN DIRECTION

- **Style**: Clean & Minimal — precise, professional, uncluttered
- **Tone**: Trustworthy, globally credible, export-focused
- **Typography**: Pair a strong, distinctive display/heading font with a clean readable body font. Avoid generic fonts like Inter, Arial, Roboto.
- **Color Palette**: Use a dominant neutral base (white or off-white) with ONE strong industrial accent color (e.g., deep navy, steel blue, or refined teal). Use the accent for CTAs, highlights, and borders.
- **Backgrounds**: Subtle texture or micro-pattern to suggest industrial precision — avoid plain flat white.
- **Animations**: Smooth, purposeful — fade-in on scroll, subtle hover states on product cards and buttons. Nothing flashy.
- **Layout**: Generous whitespace, sharp grid, clean section dividers. Every element earns its place.

---

## PAGES / SECTIONS REQUIRED

1. **Home / Hero Section** — Bold headline, trust badges, dual CTAs
2. **Products Catalogue** — 15+ product cards with filter, Request Quote button
3. **About Us / Company** — Story, strengths, stats row
4. **Export / Shipping Info** — Gujarat hub, packaging, shipping modes, incoterms
5. **Certificates & Compliance** — ISO display, REACH readiness, quality testing
6. **Contact & Inquiry Form** — Full form with validation, WhatsApp, email, location

---

## TECHNICAL REQUIREMENTS

- Single HTML file with embedded CSS and JavaScript
- Smooth scroll navigation with sticky top navbar
- Mobile-first responsive design (hamburger menu < 768px)
- Form validation with inline errors
- Google Fonts via @import with display=swap
- SEO-friendly: proper title, meta description, heading hierarchy
- No external frameworks

---

## MOBILE OPTIMIZATION

- Viewport meta tag (no user-scalable=no)
- Min 16px body font, 1.5 line height
- Min 48×48px tap targets
- CSS Flexbox/Grid with responsive breakpoints
- No horizontal scroll
- Product cards: 1 col mobile, 2 col tablet, 3-4 col desktop
- Full-width form fields on mobile
- Native `<select>` for dropdowns
- No intrusive interstitials

---

## DELIVERABLE

A single, complete, self-contained `index.html` file that is visually polished, globally credible, loads fast, and has all 6 sections with realistic placeholder content.
